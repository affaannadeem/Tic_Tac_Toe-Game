# Tic Tac Toe Game

A classic Tic Tac Toe game built in Python! This project allows two players to compete in a game of Tic Tac Toe with a simple, text-based interface. Great for learning basic programming concepts and practicing game logic.

## Features
- **Two-Player Mode**: Play with a friend in a head-to-head match.
- **Easy-to-Use Interface**: Simple text-based board for straightforward gameplay.
- **Win Detection**: Automatically checks and displays the winner after each move.
- **Tie Detection**: Identifies when the game ends in a draw.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/tic-tac-toe-game.git
    ```
2. Navigate to the project directory:
    ```bash
    cd tic-tac-toe-game
    ```
3. Run the game:
    ```bash
    python main.py
    ```

## How to Play
1. Run the game in a terminal or command prompt.
2. Players take turns entering their moves by selecting a board position (1-9).
3. The game will display the board after each move and notify players of the outcome (win, lose, or draw).
